# Luminol Project Overview

Luminol is a robust color palette generator designed for Linux desktop environments. It extracts intelligent color schemes from wallpapers to create consistent theming across desktop applications.

## Core Requirements

- **Input**: Wallpaper images (PNG, JPG, WebP) up to 10MB.
- **Output**: Standardized color palettes in multiple formats (hex, rgba, etc.).
- **Performance**: Process typical wallpapers in under 2 seconds.
- **Reliability**: Produce consistent and visually pleasing results across different image types.
